# grain-market-intervention-simulation
# 양곡 시장격리 정책 시뮬레이션 모델


## 프로젝트 개요

본 프로젝트는 양곡의 과잉 공급 상황에서 시장격리 필요 여부를 판단하고, 적정 매입 시점과 격리량을 시뮬레이션하기 위해 진행한 데이터 분석 프로젝트입니다.

쌀 생산량, 소비량, 산지쌀값, 소매가, 재배면적, 인구수, 과거 시장격리 이력 데이터를 활용하여 향후 쌀 수급 상황을 예측하고, 시장격리 정책 판단에 활용할 수 있는 분석 흐름을 구성했습니다.

---

## 전체 분석 흐름

본 프로젝트는 다음과 같은 흐름으로 진행했습니다.

1. 양곡 시장격리 제도와 문제 상황 파악
2. 생산량 예측 모델 구축
3. 소비량 예측 모델 구축
4. 초과공급 여부 판단
5. 시장격리 필요 여부 판단
6. 매입 시점 및 격리량 시뮬레이션
7. 정책 효과 검증

단순히 생산량과 소비량을 예측하는 데서 끝내지 않고, 예측 결과를 시장격리 판단 기준과 연결하는 것을 목표로 했습니다.

---

## 1. Introduction

<img width="929" height="758" alt="image" src="https://github.com/user-attachments/assets/931b39ea-0ce9-4a07-a6e1-62bd4b17ef3b" />


시장격리는 쌀 공급이 수요보다 많아 산지쌀값이 하락할 가능성이 있을 때, 정부가 일부 물량을 시장에서 분리해 가격 안정을 유도하는 정책입니다.

본 프로젝트에서는 쌀 과잉 공급이 발생했을 때 시장격리 여부를 어떻게 판단할 수 있는지, 그리고 매입 시점과 격리량을 어떻게 설정할 수 있는지를 데이터 기반으로 분석했습니다.

분석의 핵심 질문은 다음과 같습니다.

- 향후 쌀 생산량은 어떻게 변화할 것인가?
- 향후 쌀 소비량은 어떻게 변화할 것인가?
- 초과공급이 발생할 가능성이 있는가?
- 시장격리는 언제 필요한가?
- 어느 정도의 물량을 격리하는 것이 적절한가?

## 2. 생산량 예측 모델

<img width="534" height="846" alt="image" src="https://github.com/user-attachments/assets/73f76ce7-dbfb-4461-9876-512eb528c03b" />


생산량 예측에서는 과거 양곡 생산량 데이터를 활용하여 향후 생산량을 예측했습니다.

사용한 모델은 다음과 같습니다.

- ARIMA
- Prophet
- LSTM
- XGBoost
- XGBoost 스태킹 모델

생산량은 시장격리 판단의 출발점이 되는 핵심 변수입니다. 예상 생산량보다 실제 생산량이 많아질 경우 초과공급이 발생할 수 있고, 이는 산지쌀값 하락으로 이어질 가능성이 있기 때문입니다.

특히 본 프로젝트에서는 단일 모델만 사용하지 않고 여러 모델의 예측 결과를 비교했습니다. 이후 ARIMA, Prophet, LSTM의 예측 결과를 XGBoost에 입력하여 최종 생산량 예측값을 도출하는 스태킹 방식도 시도했습니다.

## 3. 소비량 예측 모델

<img width="748" height="880" alt="image" src="https://github.com/user-attachments/assets/5179ec0b-a4e3-4095-9c85-cb847ca2c816" />


소비량 예측에서는 1인당 연간 쌀 소비량 데이터를 활용했습니다.

쌀 소비량은 장기적으로 감소하는 추세를 보이기 때문에, 단순히 과거 평균을 사용하는 방식으로는 미래 수급 상황을 정확히 반영하기 어렵다고 판단했습니다.

사용한 모델은 다음과 같습니다.

- ARIMA
- VAR
- LSTM
- XGBoost
- LightGBM

특히 VAR 모델은 소비량과 생산량 간의 관계를 함께 고려하기 위해 사용했습니다. 다만 일부 모델에서는 실제 감소 추세와 맞지 않는 결과가 나타났기 때문에, 모델 결과를 그대로 사용하기보다 추세 반영 여부와 해석 가능성을 함께 고려했습니다.

## 4. 시장격리 판단 및 시뮬레이션 모델

<img width="500" height="801" alt="image" src="https://github.com/user-attachments/assets/09812f73-8a26-4927-bb75-0bc9c304b04b" />
<img width="482" height="816" alt="image" src="https://github.com/user-attachments/assets/ff915eb6-86e4-4a03-b472-f46b040a884b" />


시장격리 판단 모델에서는 생산량, 소비량, 산지쌀값, 소매가, 과거 시장격리 이력 데이터를 활용했습니다.

주요 판단 기준은 다음과 같습니다.

- 초과생산량
- 초과생산률
- 평년 수확기 산지쌀값
- 해당 연도 수확기 산지쌀값
- 평년 대비 가격 하락률
- 과거 시장격리 시행 여부
- 시장격리 매입 중량

이 기준을 바탕으로 시장격리 필요 여부를 판단하고, 추가 매입이 필요한 시점과 추천 격리량을 도출했습니다.

또한 시장격리 시행 전후의 산지쌀값 변화를 비교하여 과거 정책의 효과를 검증했습니다. 이를 통해 단순 예측 모델이 아니라, 실제 정책 판단에 가까운 시뮬레이션 구조를 만들고자 했습니다.

## 5. 분석 결과

<img width="287" height="665" alt="image" src="https://github.com/user-attachments/assets/329d68f4-c2f3-45ee-9ba3-af26e3c29597" />


분석 결과, 쌀 생산량과 소비량의 변화 추세를 함께 고려해야 초과공급 상황을 더 현실적으로 판단할 수 있음을 확인했습니다.

생산량 예측만으로는 시장격리 필요 여부를 판단하기 어렵기 때문에, 소비량 감소 추세와 산지쌀값 변화를 함께 반영하는 것이 중요했습니다.

본 프로젝트의 주요 결과는 다음과 같습니다.

- 향후 생산량 예측
- 향후 소비량 예측
- 초과생산량 및 초과생산률 계산
- 시장격리 필요 여부 판단
- 최적 매입 시점 추천
- 추천 격리량 산정
- 시장격리 정책 효과 검증

특히 예측 결과를 정책 판단 기준과 연결했다는 점에서 의미가 있습니다.

주요 역할은 다음과 같습니다.

- 분석 방향 설정

- 최종 포스터 제작

## 8. 프로젝트 의의

본 프로젝트는 단순히 머신러닝 모델을 적용하는 데서 끝나지 않고, 예측 결과를 실제 정책 판단 문제와 연결했다는 점에서 의미가 있습니다.

쌀 생산량과 소비량을 예측하고, 이를 바탕으로 시장격리 필요 여부와 매입 시점, 격리량을 판단하는 과정을 하나의 분석 흐름으로 구성했습니다.

이를 통해 농산물 수급 불균형 문제를 데이터 기반으로 해석하고, 정책 의사결정에 참고할 수 있는 시뮬레이션 모델을 구현하고자 했습니다.
