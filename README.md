## 프로젝트 개요

본 프로젝트는 양곡의 과잉 공급 상황에서 시장격리 정책의 필요 여부를 판단하고, 적정 매입 시점과 격리량을 시뮬레이션하기 위해 진행한 데이터 분석 프로젝트입니다.

시장격리는 과잉 공급된 양곡을 정부가 매입하여 시장에서 분리함으로써 가격 안정을 도모하는 정책입니다. 본 프로젝트에서는 쌀 생산량, 소비량, 산지쌀값, 소매가, 시장격리 이력 데이터를 활용하여 시장격리 정책의 효과를 분석하고, 정책 의사결정 과정을 보완할 수 있는 데이터 기반 시뮬레이션 모델을 개발하고자 했습니다.

분석은 크게 **생산량 예측 → 소비량 예측 → 시장격리 판단 → 최적 매입 시점 추천 → 시장격리량 예측**의 흐름으로 진행했습니다.

## 1. Introduction

<img width="1920" height="1005" alt="image" src="https://github.com/user-attachments/assets/4fe85b31-bdaa-4eae-8d0a-36ba768c81a0" />

양곡관리법에 따르면 시장격리란 과잉 공급된 양곡을 정부가 매입하여 시장에서 격리함으로써 가격 안정을 도모하는 정책입니다. 이는 주로 수확기에 실시되며, 농가 소득 보호에도 중요한 역할을 합니다.

매년 10월 15일에는 통계청에서 해당 연도의 예상 생산량을 발표하고, 이를 바탕으로 정부는 시장격리 여부를 포함한 관련 정책을 발표합니다. 이후 11월 15일에는 실제 생산량 통계가 발표되며, 필요한 경우 후속 조치를 통해 정책을 보완합니다.

시장격리 정책의 주요 판단 기준은 다음과 같습니다.

* 초과 생산량이 3% 이상인 경우
* 수확기 쌀 가격이 평년 대비 5% 이상 하락한 경우

그러나 기존 시장격리 정책은 일부 연도에서 가격 안정 효과가 충분하지 않았다는 평가를 받기도 했습니다. 또한 정책 결정 과정에서 경험적 판단에 의존하는 부분이 있어, 데이터 기반의 체계적인 예측과 시뮬레이션이 필요하다고 판단했습니다.

이에 본 프로젝트에서는 다양한 예측 모델을 활용해 생산량과 소비량을 예측하고, 산지쌀값 변화를 함께 고려하여 시장격리 필요 여부와 적정 매입 시점을 분석했습니다. 최종적으로는 향후 시장격리량까지 예측하여 정책 의사결정에 참고할 수 있는 기초 모델을 구현하는 것을 목표로 했습니다.

## 1.1 데이터 구성 및 전처리

본 프로젝트에서는 시장격리 판단에 필요한 생산, 소비, 가격, 인구, 농업 관련 데이터를 수집하여 분석에 활용했습니다.

### 사용 데이터

| 데이터               |        기간 | 활용 목적                 |
| ----------------- | --------: | --------------------- |
| 1인당 연간 쌀 소비량      | 1963~2023 | 소비량 예측 모델 학습          |
| 총 생산량 데이터         | 1992~2023 | 생산량 예측 및 초과공급 판단      |
| 농가 인구 데이터         | 2010~2023 | 생산량 변화 설명 변수          |
| 총인구수 데이터          | 1992~2023 | 소비량 및 생산량 변화 보조 변수    |
| 재배면적 데이터          | 1992~2023 | 생산량 예측 변수             |
| 이삭수·제현율 기반 풍흉작 지표 | 2010~2023 | 생산성 및 품질 관련 변수        |
| 산지쌀가격 동향          | 2013~2024 | 시장격리 효과 검증 및 매입 시점 분석 |
| 소매가격 데이터          | 2014~2024 | 가격 흐름 비교 및 보조 분석      |
| 시장격리 이력 데이터       | 2014~2024 | 정책 시행 여부 및 격리량 예측     |

### 전처리 과정

산지쌀값과 소매가격 데이터는 월별 5일, 15일, 25일 기준으로 제공되어 있었기 때문에, 이를 날짜 형식으로 변환한 뒤 시계열 분석이 가능하도록 정리했습니다.

주요 전처리 과정은 다음과 같습니다.

* 날짜 형식 변환
* 산지쌀값과 소매가격 데이터 통합
* wide format 데이터를 long format으로 변환
* 쉼표가 포함된 숫자형 데이터 정리
* 결측치 처리
* 연도별 데이터 병합
* 모델 학습을 위한 정규화 및 스케일링
* 시계열 모델 적용을 위한 차분 처리

이 과정을 통해 생산량, 소비량, 가격, 시장격리 이력 데이터를 하나의 분석 흐름에서 활용할 수 있도록 구조화했습니다.

## 2. Model - Production(LSTM)

<img width="591" height="376" alt="image" src="https://github.com/user-attachments/assets/9e0190f9-bdd6-4994-9e2f-42ad5610b3da" />


생산량 예측 모델은 2010년부터 2023년까지의 농업 데이터를 활용해 구축했습니다. 생산량은 시장격리 판단의 출발점이 되는 변수이므로, 향후 생산량을 예측하는 과정이 필요했습니다.

본 모델에서는 생산량 예측을 위해 다음 변수를 사용했습니다.

* 면적
* 농가 수
* 농가인구
* 풍흉작 지표
* 과거 생산량

데이터는 Min-Max 스케일링을 통해 정규화한 뒤 LSTM 모델에 입력했습니다. LSTM 모델은 시계열 데이터의 패턴을 학습하는 데 적합한 모델이기 때문에, 연도별 생산량 변화와 농업 관련 변수의 흐름을 반영하는 데 사용했습니다.

모델 학습에서는 과거 3개년 데이터를 기반으로 다음 시점의 생산량을 예측하는 Time Step 구조를 사용했습니다. 2010년부터 2020년까지의 데이터를 학습에 활용하고, 2020년부터 2023년까지를 기반으로 검증했습니다.

학습 결과, 손실 함수는 초기에 급격히 감소한 뒤 안정화되었으며, 평가 결과 MAE는 약 84,117톤, R²는 0.82로 나타났습니다. 이는 모델이 데이터 변동성을 일정 수준 설명하고 있음을 의미합니다.

최종적으로 2025년부터 2029년까지의 생산량을 예측했으며, 이 결과는 이후 소비량 예측 및 시장격리 판단을 위한 기초 자료로 활용했습니다.

## 3. Model - Consumption(VAR)

<img width="864" height="461" alt="image" src="https://github.com/user-attachments/assets/a2ff2b01-339a-45b9-8d84-4a8d64bc1d69" />

소비량 예측은 1인당 연간 쌀 소비량을 대상으로 진행했습니다. 시장격리는 단순히 생산량만으로 결정되는 것이 아니라, 소비량과의 관계를 함께 고려해야 하기 때문입니다.

본 프로젝트에서는 소비량 예측을 위해 VAR 모델을 사용했습니다. VAR 모델은 여러 시계열 변수 간의 상호작용을 동시에 고려할 수 있는 다변량 시계열 모델입니다.

1992년부터 2023년까지의 데이터를 기반으로 학습을 진행했으며, LSTM 모델로 예측한 생산량 데이터를 추가하여 2025년부터 2029년까지의 소비량을 예측했습니다.

모델링 과정에서는 다음 과정을 거쳤습니다.

* Min-Max 스케일링
* 차분을 통한 정상성 확보
* AIC 기준 최적 시차 설정
* VAR 모델 학습
* 잔차 검정을 통한 모델 안정성 확인

최종 예측 결과, 1인당 쌀 소비량은 2025년 약 55.7kg에서 2029년 약 51.1kg까지 지속적으로 감소하는 경향을 보였습니다.

이는 과거 소비량 감소 추세를 반영한 결과로, 향후 시장격리 판단에서는 생산량뿐 아니라 소비량 감소에 따른 수급 불균형 가능성도 함께 고려해야 함을 보여줍니다.

## 4. Model - Market Intervention

<img width="724" height="444" alt="image" src="https://github.com/user-attachments/assets/e147f425-863a-4cbc-bf68-ddf131a1f960" />

시장격리 모델에서는 산지쌀값을 주요 가격 지표로 사용했습니다. 산지쌀값은 농가 수취가격으로, 생산자 보호와 가격 안정을 목표로 하는 시장격리 정책의 핵심 지표입니다.

본 프로젝트에서는 산지쌀값이 정부 매입 시점에 어떻게 반응하는지 분석하고, 정책 효과를 평가하는 데 중점을 두었습니다.

---

### 4.1 시장격리 성과 검증

<img width="785" height="416" alt="image" src="https://github.com/user-attachments/assets/42523e1e-3c62-4e4d-af81-a16966ed99dd" />


먼저 현재까지의 시장격리 성공 여부를 검증했습니다. 시장격리 매입 시점이 목표 가격 상승률을 달성했는지 확인하기 위해, 격리 전후 가격 변화율을 비교했습니다.

검증 기준은 다음과 같습니다.

* 격리 전 3개월 평균 가격
* 격리 후 2개월 평균 가격
* 가격 변화율
* 목표 가격 상승률 3%

2차 시장격리가 존재하는 경우에는 1차 매입 시점 이전 3개월 평균 가격과 2차 매입 시점 이후 2개월 평균 가격을 비교하여 정책의 누적 효과를 평가했습니다.

분석 결과, 2014년부터 2016년까지는 가격 변화율이 음수로 나타나 정책 효과가 미흡했음을 확인했습니다. 반면 2017년과 2022년에는 각각 14.96%, 11.30%로 목표치인 3%를 크게 상회하며 성공적으로 작동했습니다.

2024년은 2.30%로 목표치에는 미달했지만, 상승 효과는 관측되었습니다. 이는 시장격리 매입 시점이 정책 목표 달성에 일부 기여했으나, 시기적 조정이나 추가 매입이 필요할 수 있음을 보여줍니다.

---

### 4.2 최적 매입 시점 추천 모델

시장격리 정책은 시행 여부뿐만 아니라 매입 시점이 중요합니다. 매입 시점이 늦을 경우 이미 산지쌀값이 크게 하락한 뒤일 수 있고, 너무 이른 시점에 매입할 경우 정책 효과가 충분히 나타나지 않을 수 있습니다.

본 모델에서는 정부 기준을 반영하여 산지쌀값이 평년 수확기 가격의 95% 이하로 하락하는 시점을 기준으로 최적 매입 시점을 추천했습니다.

시장격리가 시행된 연도에서는 1차 매입 시점 이후와 2차 매입 시점 이후의 산지쌀값 흐름을 각각 분석했습니다. 시장격리가 시행되지 않은 연도에서는 산지쌀값이 기준 이하로 하락하는 첫 번째 시점을 찾아 최적 매입 시점으로 추천했습니다.

분석 결과, 일부 연도에서는 추가 매입 시점이 도출되었습니다. 특히 2015년, 2016년, 2022년, 2024년에는 추가 매입 시점이 확인되었고, 2024년에는 1차와 2차 매입 모두 추가 매입이 필요하다는 결과가 나타났습니다.

이는 기존 시장격리 시점이 일부 연도에서는 충분히 효과적이지 않았을 수 있으며, 산지쌀값 하락을 완화하기 위해 보다 정교한 예측 모델이 필요함을 시사합니다.

### 4.3 시장격리량 예측 모델

<img width="837" height="411" alt="image" src="https://github.com/user-attachments/assets/31d29e8c-e9a7-43b0-ac47-c6cf1abb9b30" />

시장격리량은 생산량, 소비량, 초과생산량, 과거 시장격리 여부 등 여러 요인의 영향을 받습니다. 이러한 변수 간 관계는 단순한 선형 관계가 아니기 때문에, 본 프로젝트에서는 비선형적 관계와 변수 간 복잡한 상호작용을 반영할 수 있는 Random Forest 모델을 사용했습니다.

모델은 2014년부터 2024년까지의 데이터를 기반으로 학습했습니다. 이후 앞에서 예측한 2025년부터 2029년까지의 생산량과 소비량 예측값을 활용하여 향후 시장격리량을 예측했습니다.

사용한 주요 변수는 다음과 같습니다.

* 실제 생산량
* 예상 생산량
* 시장격리량
* 초과생산량
* 1인당 소비량
* 시장격리 시행 여부
* 초과생산 비율
* 시장격리 비율
* 생산량 및 소비량 변화율

모델 성능 평가 결과, R²는 약 0.93으로 나타났으며, MAE는 약 31,150톤, MAPE는 약 12.49%, RMSE는 약 37,284톤으로 확인되었습니다.

변수 중요도 분석에서는 시장격리 비율, 시장격리 실행 여부, 초과생산 비율이 중요한 변수로 나타났습니다. 이는 향후 시장격리량 결정에서 과거 시장격리 실적과 초과생산 관련 지표가 핵심적인 역할을 한다는 점을 보여줍니다.

## 5. Results

본 연구는 벼 수확에서 실제 시장격리 정책의 효과를 분석하고, 정책 의사결정 과정을 보완할 수 있는 시뮬레이션 모델을 개발하고자 했습니다.

분석 결과, 시장격리 정책은 일부 연도에서 산지쌀값 안정에 기여했지만 모든 상황에서 동일하게 효과적이지는 않았습니다. 특히 매입 시점과 매입량에 따라 정책 효과가 달라질 수 있음을 확인했습니다.

본 프로젝트의 주요 결과는 다음과 같습니다.

| 구분       | 내용                                         |
| -------- | ------------------------------------------ |
| 생산량 예측   | LSTM 모델을 활용하여 2025~2029년 생산량 예측            |
| 소비량 예측   | VAR 모델을 활용하여 2025~2029년 1인당 쌀 소비량 감소 추세 예측 |
| 정책 효과 검증 | 시장격리 전후 산지쌀값 변화율을 비교하여 성공 여부 검증            |
| 최적 매입 시점 | 평년 수확기 가격의 95% 이하 하락 시점을 기준으로 추가 매입 시점 추천  |
| 시장격리량 예측 | Random Forest 모델을 활용하여 향후 시장격리량 예측         |

이 모델을 통해 다양한 시장격리 시나리오에 따른 가격 변동을 시각화하고 살펴볼 수 있었습니다. 또한 정책 의사결정 과정에서 참고할 수 있는 기초적인 분석 도구로서의 가능성을 확인했습니다.

다만 본 연구는 미래 환경 변화를 모두 반영하지 못한다는 한계가 있습니다. 특히 산지 유통시장에서 중요한 역할을 하는 판매상인의 심리적 요인을 모델에 반영하지 못했으며, 복잡한 시장 메커니즘을 완전히 구현하기에는 기술적 한계가 있었습니다.

그럼에도 불구하고 본 프로젝트는 시장격리 정책의 효과를 시뮬레이션을 통해 분석하고, 데이터 기반 정책 의사결정의 가능성을 제시했다는 점에서 의미가 있습니다.

## 9. 한계점 및 개선 방향

본 프로젝트는 제한된 데이터와 학부 프로젝트 수준의 분석 환경에서 진행되었기 때문에 몇 가지 한계가 있습니다.

### 한계점

* 데이터 기간이 길지 않아 일부 모델의 일반화 성능이 제한됨
* 기상 여건, 국제 곡물 가격, 환율 등 외부 변수를 충분히 반영하지 못함
* 산지 유통인의 판매 심리와 같은 시장 참여자 행동 요인을 반영하지 못함
* 정책 시행 시점과 가격 변화 사이의 인과관계를 완전히 분리하기 어려움
* 일부 딥러닝 및 앙상블 모델은 데이터 규모에 비해 복잡하여 과적합 가능성이 존재함

### 개선 방향

* 기상 데이터, 수입쌀 가격, 국제 곡물 가격 등 외부 변수 추가
* 더 긴 기간의 생산량 및 가격 데이터 확보
* 시장격리 정책 시행 전후의 인과 효과 분석 강화
* 모델별 하이퍼파라미터 튜닝 고도화
* 가격 예측 모델과 시장격리량 예측 모델의 통합
* 실제 정책 시뮬레이션을 위한 대시보드 형태로 확장

향후 더 많은 변수와 정교한 모델을 결합한다면, 시장격리 정책의 효과성을 높이고 보다 과학적인 정책 의사결정을 지원하는 데 활용될 수 있을 것으로 기대됩니다.
