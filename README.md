## 프로젝트 개요

본 프로젝트는 양곡의 과잉 공급 상황에서 시장격리 정책의 필요 여부를 판단하고, 적정 매입 시점과 격리량을 시뮬레이션하기 위해 진행한 데이터 분석 프로젝트입니다.

시장격리는 과잉 공급된 양곡을 정부가 매입하여 시장에서 분리함으로써 가격 안정을 도모하는 정책입니다. 본 프로젝트에서는 생산량, 소비량, 산지쌀값, 시장격리 이력 데이터를 활용하여 시장격리 정책의 효과를 분석하고, 의사결정 과정을 보완할 수 있는 시뮬레이션 모델을 개발하고자 했습니다.

## 1. Introduction

시장격리는 양곡관리법에 명시된 정책으로, 과잉 공급된 양곡을 정부가 매입하여 시장에서 격리함으로써 가격 안정을 도모하는 제도입니다. 주로 수확기에 시행되며, 농가 소득 보호에도 기여합니다.

매년 10월 15일에는 통계청에서 해당 연도의 예상 생산량을 발표하고, 이를 바탕으로 정부가 시장격리 여부 및 관련 정책을 함께 발표합니다. 이후 11월 15일에는 실제 생산량 통계가 발표되며, 필요한 경우 후속 조치가 이루어집니다.

시장격리의 주요 기준은 다음과 같습니다.

- 초과 생산량이 3% 이상인 경우
- 수확기 쌀 가격이 평년 대비 5% 이상 하락한 경우

본 연구에서는 이러한 기준을 바탕으로 시장격리 정책의 효과를 분석하고, 의사결정 과정을 보완할 수 있는 시뮬레이션 모델을 개발하고자 했습니다.

## 8. 프로젝트 의의

본 프로젝트는 단순히 머신러닝 모델을 적용하는 데서 끝나지 않고, 예측 결과를 실제 정책 판단 문제와 연결했다는 점에서 의미가 있습니다.
